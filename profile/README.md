# Transmit Software — Portable FTP, SFTP, and Cloud Storage Client from Panic

## About Transmit Software

Transmit Software provides a professional file transfer client from Panic, supporting FTP, SFTP, WebDAV, and major cloud storage services. Transmit Software connects to Amazon S3, Backblaze B2, Google Drive, Microsoft OneDrive, Dropbox, and Box. Transmit Software combines a dual‑panel interface, folder synchronization, remote file browsing, cloud mounting, and transfer queues. Transmit Software integrates tabbed connections, history sync, and keyboard shortcuts directly into the interface. Transmit Software enables users to transfer files between local, remote, and cloud locations, sync folders, edit remote files, and mount clouds as local disks. Transmit Software runs on portable storage, USB drives, and local user folders. Transmit Software is trusted by web developers, designers, system administrators, cloud engineers, and creative professionals worldwide.
